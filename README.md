# Facturacion

Programa de facturación desarrollado en IntellijIdea, emplendo Java y que se muestra por consola. Consta de un menu que ofrece múltiples opciones 

Este programa está estructurado según el modelo MVC (Modelo-Vista-Controlador). 

Controlador - Esta clase gestiona todas las funciones del programa, las cuales estan repartidas entre las clases Vista y Modelo
